## 🚀 Overview

This repository contains the source code for my professional service-based portfolio. Designed to bridge the gap between aesthetic excellence and technical precision, the site showcases my expertise in **Agentic AI Workflows**, **Frontend Development**, and **UI/UX Design**. 

Whether it's crafting immersive web experiences, developing intelligent AI-driven applications, or designing user-centric interfaces, this platform highlights the comprehensive services and solutions I deliver to clients.

## ✨ Key Features

- **Immersive Design System:** Pure black aesthetic with `mix-blend-mode` effects, stark typography, and micro-interactions.
- **Smooth Navigation:** Global Lenis smooth-scrolling combined with Framer Motion `<AnimatePresence>` page transitions.
- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewing.
- **Client Communication Backend:** Integrated serverless contact form powered by Vercel API Routes and the Resend SDK to handle client inquiries securely.

## 🛠️ Tech Stack

- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion & Lenis Scroll
- **Icons:** Lucide React
- **Backend / Email:** Vercel Serverless Functions (`api/`) & Resend
